# Node Version: 18.16.0

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Test the app

```bash
yarn test
```

## Development Server

Start the development server on `http://localhost:3000`

```bash
yarn dev
```

## Production

Build the application for production:

```bash
yarn build
```

Locally preview production build:

```bash
yarn preview
```
